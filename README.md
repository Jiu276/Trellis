# Trellis - Premium Lifestyle Blog

A modern, luxurious blog website built with React, featuring product reviews, platform comparisons, and lifestyle content.

## Features

- **Elegant Landing Page**: Hero section with beautiful background imagery and smooth animations
- **Dynamic Blog System**: 10 curated articles with categories, search, and pagination
- **Product Showcase**: Premium product reviews with detailed specifications and ratings
- **Responsive Design**: Fully mobile-responsive layout that works on all devices
- **Interactive Elements**:
  - Category filtering and search functionality
  - Social media widget with working external links
  - Back-to-top button for easy navigation
  - Smooth animations and transitions throughout
- **Rich Content Pages**:
  - Detailed article pages with related content
  - Product comparison and reviews
  - About page with company values and stats
  - Contact form with FAQs

## Tech Stack

- **React 18**: Modern UI library
- **React Router**: Client-side routing
- **Vite**: Fast build tool and dev server
- **CSS3**: Custom styling with CSS variables and animations
- **Unsplash**: High-quality product and lifestyle images

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```
